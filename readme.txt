== Description ==

This custom field accepts a CSV file and converts the data to JSON. Use `get_field()` to retrieve the raw JSON.

= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-csv-to-json` folder into your `wp-content/plugins` folder
2. Activate the CSV to JSON plugin via the plugins admin page
3. Create a new field via ACF and select the CSV to JSON type
4. Read the description above for usage instructions
